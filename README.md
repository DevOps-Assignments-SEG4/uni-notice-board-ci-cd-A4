# University Digital Notice Board

Static website with CI pipeline enforcing linting, build, and Dockerization.

## Tech Stack
- HTML / CSS
- Parcel
- GitHub Actions
- Docker / Nginx

## CI Pipeline And CD pipline
1. HTMLHint
2. Stylelint
3. Parcel Build
4. Docker Image Build & Push
5. initial deploye menual on the render.com
6. then make the automation so that when any merging done CD pipline will fire.
